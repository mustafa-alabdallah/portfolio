How this site is hosted (free)

  1. Create a free GitHub account & new public repo, e.g., portfolio.
  2. Add this index.html file at the repo root. Create folders like /assets, /games, /presentations, and upload your files.
  3. In the repo: Settings → Pages → Build from Deploy from a branch, select main and / (root). Save.
  4. Your site will be live at https://<your-user>.github.io/portfolio/. Optional: add a custom domain in Pages.

Alternative free hosts: Cloudflare Pages, Netlify, Vercel — all work with static sites (no server needed).
